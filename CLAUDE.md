# ink-markdown-es

## Tech

- Ink
- Marked

## Package Manager

- bun

## Main Thread

- Use marked to parse markdown
- Use marked-terminal to render markdown in ink Text
- Use highlight.js to highlight code
- Use memo to cache rendered output

## Scripts

- `bun run dev` to run the example
- `bun run build` to build the package
- `bun run format` to format the code
- `bun run lint` to lint the code
